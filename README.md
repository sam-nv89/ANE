# Adaptive Nutrition Engine (ANE) - SPA

**Adaptive Nutrition Engine** — это интеллектуальный сервис персонализированного питания, объединяющий точный научный расчет КБЖУ, учет реального тайм-менеджмента пользователя и его вкусовой комфорт.

Изначально проект стартовал как прототип лендинга для инновационного алгоритма питания, но затем эволюционировал в полноценное **Single Page Application (SPA)**, покрывающее весь цикл: от первоначального опроса (онбординга) до генерации меню на неделю, списка покупок и отслеживания прогресса.

## 🎯 Ценностное предложение
Алгоритм питания, который подстраивается под вашу жизнь, а не наоборот. Система автоматически формирует рацион, учитывая биометрию, заболевания, время на готовку и вкусовые предпочтения, а также минимизирует монотонность блюд.

## 🚀 Основной функционал (MVP)
- **Умный онбординг (6 шагов):** Сбор биометрических данных, постановка целей (похудение/поддержание/набор), временные ограничения, вкус/аллергены.
- **Математика КБЖУ:** Автоматический расчет суточной нормы потребления калорий и БЖУ по формуле Миффлина-Сан Жеора.
- **Генератор меню:** Формирование 7-дневного плана (завтрак, обед, ужин, перекус) на основе гибких алгоритмов подбора рецептов (с fallbacks и динамическими окнами).
- **Интерактивный дашборд:** Наглядный план питания с кольцами прогресса макронутриентов.
- **Zero-Waste Shopping List:** Автоматически сформированный список покупок, агрегирующий ингредиенты из рецептов на всю неделю без остатков.
- **Трекинг прогресса:** Отслеживание изменений веса и соблюдения плана.
- **Управление профилем:** Возможность изменения ключевых параметров с немедленным пересчетом плана питания.

## 🛠 Технологический стек
- **Frontend Framework:** React 19 + Vite 8
- **Роутинг:** React Router v7
- **Стейт-менеджер:** Zustand + persist (локальное хранилище)
- **Анимации:** Framer Motion
- **UI/Стилизация:** Индивидуальный дизайн (Vanilla CSS Variables). Гамма: Electric Green (`#00f5a0`), Cyan (`#00d4ff`), Violet (`#7c3aed`), Dark theme.
- **Иконки:** Lucide React

## 📦 Установка и запуск

1. Склонируйте репозиторий проекта.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```
4. Для сборки production версии:
   ```bash
   npm run build
   ```
